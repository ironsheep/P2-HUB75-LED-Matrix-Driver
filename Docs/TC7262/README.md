# TC7262 - LED Display Line Driver Chip

**Manufacturer:** SHENZHEN FUMAN ELECTRONICS CO., LTD.

## Description
8-channel LED display line scanning row driver with integrated 138 decoder and power PMOS outputs.

## Key Specifications
- Package: SOP-16
- Integrated 138 decoding circuit
- 8 power PMOS outputs (2.5A max, 120mΩ on-resistance)
- Supports 1-8 scan patterns
- Anti-ghosting features
- Max power: <625mW @ VDD=5V
